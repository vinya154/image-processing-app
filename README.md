# Image Processing Web Application

This project is an interactive web application that allows users to upload images and apply different image processing filters.

## Features
- Upload an image
- Apply filters like Blur, Detail, and Grayscale
- View processed output instantly

## Technologies Used
- Python
- OpenCV
- Gradio

## How it works
The user uploads an image through a simple interface. The system processes the image using OpenCV and displays the output based on selected filters.
